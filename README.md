# AC Shopping Cart - darkmode
```
購物車切版。
```
##功能

1. RWD 設計與網頁版
2. 手機版有漢堡排
3. 點擊下一步或上一步切換流程的顯示
4. 可以切換白天跟黑暗模式
5. 使用webpack把scss檔案，依照不同區塊進行分割

## 環境

1. npm 6.14.15
2. webpack 5.65.0
3. mini-css-extract-plugin: 2.4.6
